# Umme Farhana Sumaiya | Portfolio

A modern, dynamic portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Deployment**: Vercel

## Features

- ✨ Smooth animations with Framer Motion
- 🎨 Dark theme with gradient accents
- 📱 Fully responsive design
- 🧊 Glass morphism UI elements
- ⚡ Particle background animation
- 📊 Interactive skill display
- 📝 Contact form with status feedback
- 🚀 API route for traffic data demo

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Open http://localhost:3000
